字母對應表Substitution mapping
凱撒密碼把每個字母沿字母表位移固定格數。它是替換式密碼的例子,但金鑰只有 25 種可能,用暴力破解試盡所有位移便能輕易破解,故不安全。現代加密用極大的金鑰空間。 The Caesar cipher shifts each letter a fixed number of places along the alphabet. It is a substitution cipher, but with only 25 possible keys a brute-force attack trying every shift cracks it instantly — so it is insecure. Modern encryption uses an astronomically large key space.
