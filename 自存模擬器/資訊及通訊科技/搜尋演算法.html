線性搜尋由頭逐個檢查,最多要看 n 個 → O(n),任何陣列都適用。二分搜尋每次看中間值,把範圍對半切,只需約 log₂n 步 → O(log n),但陣列必須先排序。 Linear search checks each item from the start — up to n looks → O(n), works on any array. Binary search checks the middle and halves the range each time — about log₂n steps → O(log n), but the array must be sorted first.
