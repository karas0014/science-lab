字元編碼探索Character Encoding Explorer

ICT · ASCII / Unicode
字符集大小 ↔ 位元數Charset size ↔ number of bits
bits
ASCII — 7 位元，128 個字符，只含英文字母、數字及符號。7-bit, 128 characters: English letters, digits, symbols.
Big-5 / GB — 雙位元組區域標準；Big-5 收繁體字，GB 收簡體字。Double-byte regional standards; Big-5 for Traditional, GB for Simplified Chinese.
Unicode — 統一涵蓋全球文字，UTF-8 以 1–4 位元組儲存。Universal for all scripts; UTF-8 stores each char in 1–4 bytes.
亂碼 — 若用錯編碼開檔 (如 Big5 檔以 GB 開)，就會出現亂碼。簡體「爱」在 Big5 沒有碼位。Opening a file with the wrong encoding causes 亂碼 (mojibake). Simplified 爱 has no Big-5 code point.
