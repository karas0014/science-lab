溢出 (Overflow)： 當運算結果超出 n 個位元能表示的範圍，最高位的進位會被丟棄，儲存的數值便會出錯。二補數 (two's complement) 用最高位作正負號，讓同一套加法電路可處理正負數。 Overflow: when a result exceeds the range n bits can hold, the top carry is discarded and the stored value is wrong. Two's complement uses the top bit as a sign, so one adder circuit handles both positive and negative numbers.
