真值表Truth table
布林邏輯只有兩個值:真 (1) 與假 (0)。AND 全真才真;OR 有一真即真;NOT 反轉。程式的選擇 (IF) 與迭代 (WHILE) 都靠布林條件決定走向。真值表列出所有輸入組合的輸出。 Boolean logic has just two values: true (1) and false (0). AND is true only if all inputs are true; OR is true if any is; NOT inverts. Program selection (IF) and iteration (WHILE) both branch on Boolean conditions. A truth table lists the output for every input combination.
