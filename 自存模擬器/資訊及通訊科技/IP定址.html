IP 位址是每部連網裝置的唯一識別。IPv4 用 4 個 0–255 的位元組 (共 32 位元);由於裝置暴增,IPv4 已不敷應用,IPv6 用 128 位元提供近乎無限的位址。私有位址 (如 192.168.x.x) 不能直接上網,需經 NAT 轉換。 An IP address uniquely identifies each networked device. IPv4 uses four 0–255 octets (32 bits total); with the explosion of devices it ran out, so IPv6 uses 128 bits for a near-unlimited supply. Private addresses (e.g. 192.168.x.x) can't reach the internet directly and go through NAT.
