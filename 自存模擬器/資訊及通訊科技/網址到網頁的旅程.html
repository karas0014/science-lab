網址到網頁的旅程URL to Web Page Journey

ICT · DNS · HTTP · HTTPS
DNS 把網域名稱翻譯成 IP 位址;HTTP/HTTPS 是瀏覽器與伺服器溝通的協定 (HTTPS 加密);瀏覽器發出 GET 請求,伺服器回傳 HTML,再由瀏覽器渲染成畫面。 DNS translates a domain name into an IP address; HTTP/HTTPS is the protocol the browser and server talk with (HTTPS is encrypted). The browser sends a GET request, the server returns HTML, and the browser renders it.
