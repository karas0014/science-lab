陣列把同類型的多個資料放在一個名字下,用索引存取個別元素。索引通常由 0 開始,所以 6 個元素的索引是 0–5。要處理每個元素,就用迴圈遍歷 (traverse) 整個陣列。 An array stores many values of the same type under one name, reaching each element by its index. Indices usually start at 0, so 6 elements have indices 0–5. To process every element you traverse the array with a loop.
