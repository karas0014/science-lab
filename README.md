# 曾璧山(崇蘭)中學 · 科學組互動學習平台

A static website of interactive science & ICT simulators, self-hosted for
Tsang Pik Shan (Sung Lan) Secondary School's Science Education Section.

## Structure
```
index.html            主頁（5 科入口）
physics.html          物理（42 個模擬器）
chemistry.html        化學（18 個）
biology.html          生物（10 個）
science.html          綜合科學（8 個）
ict.html              資訊及通訊科技（暫未有模擬器）
自存模擬器/            所有模擬器 HTML（自存，離線可用）
```

All simulators are self-contained (no external servers, no data collection).
Adapted with permission from the 青松中學 and phcsy science teams.

## Live site
Enable **GitHub Pages** (Settings → Pages → Deploy from branch → `main` / root),
then the site is served at `https://<user>.github.io/<repo>/`.
